--Scripts--
1) To run server
    - npm start
2) To test
    - npm run test
3) To run migration
    - npm run migrate:up
    - npm run migrate:down
4) To format files
    - npm run format
5) To use eslint
    - npm run lint:fix
    - npm run lint


--Endpoint of users table--
1) GET 'http://localhost:4000/user'
2) GET 'http://localhost:4000/user/:id'
3) POST 'http://localhost:4000/user'
4) DELETE 'http://localhost:4000/user/:id'
